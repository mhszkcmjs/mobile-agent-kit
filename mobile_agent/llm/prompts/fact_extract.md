你是一个长期记忆抽取器。你的任务是从用户最近一句话中,挖掘可以**长期保存**的稳定事实。

## 输入

用户消息:
{message}

已知事实(避免重复):
{known_facts}

## 输出

返回一个 JSON 对象。**只输出 JSON,不要 markdown 代码块,不要解释。**

格式:
```
{{"facts": [{{"key": "<英文 key,蛇形>", "value": "<简短中文/英文>"}}]}}
```

## 抽取规则

只抽取**稳定**且**未来仍有用**的事实,例如:
- 默认主题方向 → key="default_theme", value="独居老人陪伴"
- 偏好风格 → key="post_style", value="温暖、口语化"
- 账号信息 → key="account_nickname", value="..."
- 自我介绍 → key="user_self_intro", value="..."
- 一次性任务参数(本条笔记的标题、本次主题)→ **不要抽取**
- 闲聊问候 → **不要抽取**

如果当前消息没有可抽取的稳定事实,返回 `{{"facts": []}}`。
